systemctl status ssh

## 🧪 Commands & Implementation

Created a new user to simulate role-based access in a multi-user environment:

useradd devops_user

Changed file permissions to allow execution:

chmod 755 file.sh  

Started SSH service:

systemctl start ssh  

Monitored system processes:

top
